IT Asset Management – Common Sections & Fields

1.  BASIC INFORMATION

Asset Identity - assetId (system generated) - assetTag - barcode /
qrCode - assetName - assetDescription

Asset Classification - assetCategory (Hardware / Network / Peripheral) -
assetType (Desktop / Laptop / Printer / Router etc.) - assetSubType

Manufacturer Details - brand - manufacturer - modelName - modelNumber -
partNumber - serialNumber

Asset Status - assetStatus - In Stock - In Use - Under Repair -
Disposed - Lost

Condition - assetCondition - New - Good - Fair - Damaged

Ownership - ownershipType - Owned - Leased - Rented

Dates - manufacturingDate - installationDate - endOfLifeDate

System Metadata - createdBy - createdAt - updatedBy - updatedAt

2.  PURCHASE INFORMATION

Purchase Type - purchaseType - PO - Direct

If Purchase Type = PO - poNumber - poDate - poAmount - poApprovedBy

Purchase Details - purchaseDate - vendorId - vendorName -
vendorContactPerson - vendorPhone - vendorEmail

Invoice Details - invoiceNumber - invoiceDate - invoiceAmount -
taxAmount - totalAmount - currency

Delivery Details - deliveryDate - deliveryChallanNumber - receivedBy

Documents - poDocument - invoiceDocument - deliveryChallanDocument

3.  WARRANTY INFORMATION

Warranty Available - warrantyAvailable - Yes - No

If Warranty = Yes

Warranty Mode - warrantyMode - Duration - End Date

If Mode = Duration - warrantyDuration - durationUnit - Year - Month

Warranty Dates - warrantyStartDate - warrantyEndDate

Warranty Provider - warrantyProvider - Manufacturer - Vendor - Extended
Warranty

Support Details - supportVendor - supportPhone - supportEmail

AMC / Maintenance Contract - amcAvailable - Yes - No

If AMC = Yes - amcVendor - amcStartDate - amcEndDate - amcCost

4.  ASSIGNMENT INFORMATION

Assignment Type - assignmentType - User - Location - Department

If Assignment Type = User - assignedUserId - assignedUserName -
assignedUserEmail - assignedUserDepartment

If Assignment Type = Department - assignedDepartment - departmentHead

Assignment Details - assignmentDate - expectedReturnDate -
assignmentStatus - Assigned - Returned - Lost

Asset Handover Details - handoverBy - receivedBy - handoverNotes

5.  LOCATION INFORMATION

Organization Location - organization - site / school - building -
floor - room

Asset Placement - locationType - Lab - Office - Server Room - Classroom

Rack / Network Location (optional) - rackNumber - rackUnit

6.  REPAIR & MAINTENANCE

Repair Status - repairStatus - None - Under Repair - Repaired

Repair Details - issueTitle - issueDescription - reportedDate -
reportedBy

Repair Vendor - repairVendor - vendorContact - vendorPhone

Repair Process - repairStartDate - repairEndDate - repairCost -
repairStatus

Replacement Details - partsReplaced - replacementCost

Maintenance History - lastMaintenanceDate - nextMaintenanceDate -
maintenanceNotes
